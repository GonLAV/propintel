```typescript
// דוגמה לשימוש

3. בחר מחוז (אופציונלי): "מרכז", "תל אביב", "ירושלים" וכו׳

6. תקבל עסקאות מכל הארץ עם:
   - מספר ערים שנמצאו
   - מספר מחוזות
   - מחיר ממוצע למ"ר ארצי
   - פילוח לפי עיר ומחוז
```

// דוגמה לשימוש

## 📊 סטטיסטיקות ארציות

המערכת מחשבת אוטומטית:

### נתוני שוק
- **מחיר ממוצע למ"ר** - ממוצע ארצי
- **מחיר חציוני** - median price
- **טווח מחירים** - min/max

  מחוז תל אביב: ₪27,000/מ"ר
  מחוז מרכז: ₪20,000/מ"ר
  מחוז ירושלים: ₪19,000/מ"ר

## 📋 API החדש - RealIsraeliGovDataAPI

### חיפוש עסקאות ארצי

```typescript
import { realIsraeliGovDataAPI } from '@/lib/realIsraeliGovDataAPI'

// חיפוש ארצי
const transactions = await realIsraeliGovDataAPI.searchNationalTransactions({
  cities: ['תל אביב-יפו', 'חיפה'],          // אופציונלי
  districts: ['מרכז'],                       // אופציונלי
  propertyTypes: ['דירה', 'משרד'],          // אופציונלי
  minPrice: 1000000,                        // אופציונלי
  maxPrice: 3000000,                        // אופציונלי
  minArea: 80,                              // אופציונלי
  maxArea: 120,                             // אופציונלי
  verifiedOnly: true,                       // אופציונלי
  limit: 50                                 // אופציונלי
})

// חישוב סטטיסטיקות
const stats = realIsraeliGovDataAPI.calculateNationalStatistics(transactions)

console.log(`נמצאו ${stats.totalTransactions} עסקאות`)
console.log(`מחיר ממוצע: ₪${stats.avgPricePerSqm}/מ"ר`)
console.log(`ערים: ${Object.keys(stats.byCity).length}`)
console.log(`מחוזות: ${Object.keys(stats.byDistrict).length}`)
```

### קבלת רשימת ערים
